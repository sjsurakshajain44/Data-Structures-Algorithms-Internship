# Data Structures and Algorithms Internship

This repository contains my Data Structures and Algorithms internship assignments and mini projects implemented in C++.

## Week 1 - DSA Fundamentals & Arrays

### Topics Covered

* Introduction to DSA
* Time and Space Complexity
* Big O Notation
* Arrays and Strings
* Linear Search
* Binary Search
* Bubble Sort
* Selection Sort

### Assignments

1. Linear Search
2. Binary Search on a sorted array
3. Bubble Sort with swap count
4. Reverse an array without using an extra array

### Mini Project - Student Marks Analyzer

The Student Marks Analyzer takes marks of N students as input, sorts the marks, displays the top 3 marks, and calculates the average score.
